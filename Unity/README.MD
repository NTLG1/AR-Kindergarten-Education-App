## 🛠 Prerequisites & Setup

This project requires the **Vuforia Engine** package. Please follow these steps to set up the project:

1. **Download the Vuforia Tarball:**
   - Download `com.ptc.vuforia.engine-10.25.4.tgz` from the [Vuforia Developer Portal](https://developer.vuforia.com/downloads/sdk).
2. **Place the file:**
   - Create a folder named `Packages` in the project root (same level as `/Assets`).
   - Move the downloaded `.tgz` file into that folder.
3. **Link in Unity:**
   - Open the project in Unity. 
   - Go to `Window > Package Manager`.
   - Click the **+** icon -> **Add package from tarball...**
   - Select the file in the `LocalPackages` folder.
